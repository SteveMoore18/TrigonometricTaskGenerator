# Trigonometric-Task-Generator
Генератор тригонометрических выражений

# Install on Linux

First you need a Python 3.6.8

`sudo apt-get install python3`

Next install a pip3

`sudo apt-get install python3-pip`

And install a PySide2

`pip3 install PySide2`

That's all!

Run `main.py`

`python3 main.py`
